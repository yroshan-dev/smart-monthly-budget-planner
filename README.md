# 💰 Smart Monthly Budget Planner

An interactive personal finance web application that helps users understand spending habits, monitor savings, and evaluate overall financial health through real-time calculations and visual analytics.

## 🌐 Live Demo

🔗 https://yroshan-dev.github.io/smart-monthly-budget-planner/

## 📌 Features

- Real-time budget calculations
- Monthly income and expense tracking
- Savings rate analysis
- Financial health scoring system
- Expense breakdown visualization
- Interactive budget presets
  - 50/30/20 Rule
  - High Saver
  - Debt Heavy
- Responsive design for desktop and mobile
- Modern dark-themed interface
- INR (₹) currency support

## 📊 Key Metrics

The application automatically calculates:

- Total Monthly Income
- Total Expenses
- Remaining Balance
- Savings Rate
- Financial Health Score

## 🎯 Budget Presets

### 50/30/20 Rule
A balanced budgeting strategy allocating:
- 50% Needs
- 30% Wants
- 20% Savings

### High Saver
Designed for users focused on wealth accumulation and long-term financial goals.

### Debt Heavy
Optimized for users prioritizing debt repayment while maintaining essential expenses.

## 🛠️ Built With

## 🚀 Local Development

Clone the repository:

```bash
git clone https://github.com/yroshan-dev/smart-monthly-budget-planner.git
```

Navigate into the project:

```bash
cd smart-monthly-budget-planner
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
